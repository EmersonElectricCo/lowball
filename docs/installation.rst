###################
Installing lowball
###################

lowball has been tested to work with only Python 3.6+

Pip
***

.. code-block:: bash

    pip install lowball

From Source
***********

.. code-block:: bash

    git clone https://github.com/EmersonElectricCo/lowball
    cd ./lowball
    pip install -r requirements.txt
    python setup.py install
